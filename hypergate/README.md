# HyperGate

Smartest IPFS gateway ever.


# FAQ

## API

```
- /ipfs/:id - get asset by IPFS hash
```

## Dev

1. Instal `wrangler` CLI V2

```bash
npx wrangler
```

2. Login

```bash
npx wrangler login
```

3. Run dev

```bash
npx wrangler dev
```


9999. Publish

```bash
npx wrangler publish
```

## Caveats

1. Make sure you setup correct `account_id` in `wrangler.toml`
